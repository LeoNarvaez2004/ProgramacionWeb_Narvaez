# Programación Web — Ejercicios y Prácticas de Clase

**Autor:** Leonardo Narváez  
**Institución:** Universidad de las Fuerzas Armadas ESPE  
**Materia:** Programación Web  

---

## Descripción

Repositorio de ejercicios, prácticas y talleres desarrollados durante el curso de Programación Web. Incluye el trabajo progresivo desde las bases del desarrollo frontend hasta la construcción de interfaces responsivas y estilizadas, aplicando buenas prácticas de estructura HTML y diseño con CSS.

---

## Contenido

| Unidad | Tema |
|---|---|
| Prácticas Iniciales | Estructura HTML, tablas, listas y formularios |
| Flexbox | Diseño de layouts con CSS Flexbox |
| Tarjetas (Cards) | Componentes visuales con CSS personalizado |
| CDN y Frameworks | Uso de Bootstrap vía CDN |
| Taller Formulario | Validación y diseño de formularios |
| Fondos y Secciones | Manejo de imágenes de fondo y secciones temáticas |
| Proyecto Final | Sitio web completo con múltiples páginas |
